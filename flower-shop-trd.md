# 🌸 향기나는 꽃집 TRD

Technical Requirements Document

------------------------------------------------------------------------

# 1. System Overview

향기나는 꽃집 랜딩페이지는 **정적 웹사이트 기반 예약 주문 랜딩페이지**로
제작됩니다.

사용자는 랜딩페이지에서 꽃 상품과 배송 정보를 확인한 뒤\
**Google Form을 통해 주문 예약을 접수**합니다.

주문 데이터는 **Google Sheets에 자동 저장**되며\
운영자는 시트를 통해 주문을 확인하고 **카카오톡 상담으로 주문을
확정**합니다.

# 10. Technical Constraints

본 프로젝트는 다음 기능을 포함하지 않습니다.

-   서버 사이드 개발
-   데이터베이스 구축
-   관리자 페이지
-   온라인 결제
-   자동 주문 관리
-   자동 알림 시스템

주문 관리는 **Google Sheets + 카카오톡 상담**으로 처리합니다.
